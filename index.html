Proyecto Introducción Ciencias de Datos

Contexto y Motivación

En el proyecto se intentan determinar posibles factores que determinen el precio de vehículos usados vendidos en chileautos.cl




Objetivos:

  • Determinar qué factores explican el precio de un vehículo. Para esto se entrenaron distintos modelos de regresión y se compararon para ver qué tan buenas predicciones se pueden hacer en base a los datos que disponemos
  • Estudiar la distribución de los datos según las características estudiadas



Datos

Para realizar el proyecto se extrajo información del sitio web https://www.chileautos.cl/ mediante webscrapping usando las librerias selenium y beautifulsoup. La librería requests no es capaz de obtener el código HTML de la página debido a la forma en que está diseñada la página. El proceso de webscrapping fué bastante tedioso y no se pudo extraer toda la información que buscabamos debido a qué la página detectaba que se estaba ingresando al sitio web de forma automatizada y pedía completar una prueba de "verifica que no eres un robot", la cuál no dejaba completar, ni siquiera aunque se intentara completar la prueba manualmente.

Debido a eso se pudieron extraer solo los datos de la página principal, o sea la marca, el modelo, la transmisión, el kilometraje, región del país de la publicación, tipo de vendedor, el año y el precio de cada publicación.




Análisis exploratorio de datos

Análisis Estadístico

A través de estudiar los datos agrupándolos por región y por tipo de transmisión se pudieron llegar a las siguientes conclusiones:

  • Hay una clara desigualdad en la cantidad de publicaciones, la región con más publicaciones es la Metropolitana por una gran diferencia. De hecho hay regiones que no tienen ninguna publicación, por lo menos en los modelos estudiados
  • EL promedio del precio según la región varía bastante
  • El tipo de vendedor más frecuente en la página web son vendedores particulares
  • La mayoría de los autos a la venta son mecánicos

Clustering

Para identificar grupos o distinciones entre los datos se implementó un algoritmo de clustering.

Los resultados parecen ser más que nada una disitnción por segmentos de año, lo que tiene sentido con nuestras observaciones


Regresión

Se probaron algoritmos de regresión lineal, ridge, lasso y polinomial para probar cuál se adaptaba mejor a los datos
