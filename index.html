Proyecto Introducción Ciencias de Datos

En el proyecto se intentan determinar posibles factores que determinen el precio de vehículos usados vendidos en chileautos.cl

Para esto se extrajo información del sitio web mediante webscrapping usando las librerias selenium y beautifulsoup. La librería requests no es capaz de obtener el código HTML de la página debido a la forma en que está diseñada la página. El proceso de webscrapping fué bastante tedioso y no se pudo extraer toda la información que buscabamos debido a qué la página detectaba que se estaba ingresando al sitio web de forma automatizada y pedía completar una prueba de "verifica que no eres un robot", la cuál no dejaba completar, ni siquiera aunque se intentara completar la prueba manualmente.

Debido a eso se pudieron extraer solo los datos de la página principal, o sea la marca, el modelo, la transmición, el kilometraje, el año y el precio de cada publicación.
